## Waarom hier?

- **CYD24/CYD28** zijn als doelplatform volledig uit de codebase verwijderd.
- Deze documenten zijn bewaard voor wie de verwijderstappen of de oude CYD-specifieke optimalisaties wil naslaan.
- Ze beschrijven geen actieve ondersteuning en wijken niet af van de huidige code.

Voor actuele platforminformatie: zie [00_OVERVIEW.md](../00_OVERVIEW.md), [05_CONFIGURATION.md](../05_CONFIGURATION.md) en [CODE_INDEX.md](../CODE_INDEX.md).
